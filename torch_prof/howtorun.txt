  Complete Example Run

  Here's a complete workflow:

  # 1. Go to gsplat directory
  cd /home/AMD/chanshar/workspace/gsplat

  # 2. Clean previous results (optional)
  rm -rf torch_prof/logs/* logs/*

  # 3. Inject profiler (only needed once)
  python torch_prof/profile_tests.py --inject-profiler

  # 4. Profile all tests (excluding compression)
  ENABLE_PROFILER=1 CUDA_VISIBLE_DEVICES=0 python torch_prof/profile_tests.py --tests
  test_basic.py test_rasterization.py test_2dgs.py test_strategy.py

  # 5. Profile kernels
  CUDA_VISIBLE_DEVICES=0 python torch_prof/profile_kernels.py

  # 6. Check what was profiled
  ls -la torch_prof/logs/tests/
  ls -la torch_prof/logs/kernels/

  # 7. View summary
  cat torch_prof/profiling_summary.json | python -m json.tool

  Skip Compression Tests in pytest

  If you want to run pytest directly and skip compression tests:

  # Skip tests containing "compression" in their name
  ENABLE_PROFILER=1 CUDA_VISIBLE_DEVICES=0 pytest -v -s tests/ -k "not compression"

  # Or mark compression tests to skip
  ENABLE_PROFILER=1 CUDA_VISIBLE_DEVICES=0 pytest -v -s tests/
  --ignore=tests/test_compression.py

  This will profile all the CUDA/HIP kernels and test functions while skipping the
